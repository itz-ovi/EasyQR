🚀 QR Code Generator Website

A modern, fast, and fully responsive QR Code Generator built using pure HTML, CSS, and JavaScript. This tool allows users to instantly generate QR codes for various types of data directly in the browser—no backend or database required.

---

🌟 Features

- 🔳 Generate QR codes for:
  
  - Text
  - URLs
  - Email
  - Phone numbers
  - WiFi credentials

- ⚡ Real-time QR preview

- 🎨 Customize QR color (foreground & background)

- 📏 Adjustable QR size (small, medium, large)

- 🖼️ Optional logo inside QR code

- ⬇️ Download QR code as PNG

- 📋 Copy input data instantly

- 🔄 Clear/reset input fields

- ❗ Input validation with error handling

- 🌙 Dark mode / Light mode toggle

- 💾 Save last generated QR (using localStorage)

- 🔔 Toast notifications (e.g., download success)

---
